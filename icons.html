Ícones

Utilizar os ícones é bastante simples. Para adicionar um ícone a qualquer elemento html, basta adicionar a classe icon-nome-do-icone-before. Os ícones possuem os seguintes tamanhos padrões: 8x10, 16x18 e 32x34.

Arquivo

_icons.scss

Se for necessária apenas a exibição do link, sem nenhum texto ao lado, utiliza-se a seguinte estrutura: class = 'icon-xxxx show text-replacement'. Note que é necessário adicionar três classes ao elemento.

Veja alguns exemplos de uso:

Dúvidas frequentes


<h3 class="icon-frequent-questions-blue_32_34-before">Dúvidas frequentes</h3>

Instituição 
<span class="icon-institution-blue_32_34 show text-replacement">Instituição</span>
Note que a string 'Instituição' não é exibida por conta da utilização da classe text-replacement.

Nomes

Os nomes dos ícones também possuem um padrão pré-definido para facilitar ainda mais a utilização. Todos os ícones possuem no seu nome a cor utilizada e seu tamanho em pixels.

O padrão é o seguinte: nome-do-icone_cor_altura_largura.
