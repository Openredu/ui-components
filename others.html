Auto complete

A estrutura do controle de autocomplete é a mesma dos outros elementos de controle dos formulários, com algumas adições. O .control-group deve levar .control-autocomplete. O input text deve ter .control-autocomplete-input.

Existe um caso especial do autocomplete no Redu que é usado para convidar pessoas por e-mail. Quando este caso for usado, o .control-group deve levar também .control-invite-by-mail e depois do .control-autocomplete-input, deve haver um input hidden .control-autocomplete-invites.

Arquivo

_forms.scss

Dependências
  1. vendor/jquery.tokeninput.js
  2. bootstrap-autocomplete.js

A biblioteca jQuery Tokeninput é utilizada para a funcão de auto completar.

Preenchimento do autocomplete

É aconselhável dar uma lida na documentação do Tokeninput para entender como o preenchimento do autocomplete é feito. Basicamente é um JSON com o a seguinte estrutura:

[
  {"name": "Tiago Ferreira"
  ,"id":9
  ,"avatar_32":"http://s3.amazonaws.com/redu_uploads/users/avatars/9/thumb_32/ride.png"
  ,"mail":"nomesobrenome@email.com"
  }
  ...
]

Os atributos id e name são obrigatórios. O atributo mail é obrigatório se o autocomplete de convidar por e-mail estiver sendo usado.

Ativando o autocomplete

Para ativar o autocomplete é preciso usar o seguinte script:

$('.control-autocomplete-input').tokenInput($JSON-resource)

Onde $JSON-resource aponta para o recursos JSON.

No domínio do Redu, o endereço /pessoas/auto_complete.js pode ser usado para procurar os usuários por nome.

To do: o endereço ainda não está retornando os e-mails dos usuários.

Submissão autocomplete

O input text em que o autocomplete for ativado terá como valor os ids dos itens adicionados separados por vírgula. No caso do autocomplete de convidar por e-mail, o input hidden .control-autocomplete-invites terá como valor os e-mails adicionados separados por vírgula.

Autocomplete genérico
Autocomplete com convidar 
<script>
  $('#UNIQUE_ID').tokenInput(JSON_RESOURCE)
</script>

<!-- FORM -->
<div class="control-group control-autocomplete [control-invite-by-mail]">
  <div class="control-label">
    <label for="token-input-UNIQUE_ID">LABEL_NAME</label>
  </div>
  <div class="controls">
    <input class="control-autocomplete-input" type="text" id="UNIQUE_ID">
    <!-- IF IT'S INVITE BY MAIL AUTOCOMPLETE, INSERT BELOW: -->
    <input class="control-autocomplete-invites" type="hidden">
  </div>
</div>

Barras de progresso

As barras de progresso são elementos com a classe .progress-bar-total, representando a barra total, que contém a classe .progress-bar-completed, representando a porcentagem completada.

Existem 3 tamanhos de barra, o menor leva a classe .progress-bar-mini, o médio .progress-bar-medium e maior .progress-bar-large. Além disso as barras podem ficar mais grossas adicionando a classe .progress-bar-thick.
