Mensagens do sistema (sobrepostas)

Exemplos de mensagens

Envolva sua mensagem e um ícone opcional de fechar em uma div com uma simples classe.

× Padrão. Simples 
        <div class="system-msg">
          <a data-dismiss="system-msg" class="close">×</a>
          <strong>Padrão.</strong> Simples
        </div>

Extenda facilmente a mensagem com duas classes adicionais: .msg-block para mais padding e .system-msg-heading para o cabeçalho.

×

Padrão.

Simples 
        <div class="system-msg system-msg-block">
          <a data-dismiss="system-msg" class="close">×</a>
          <h4 class="system-msg-heading">Padrão.</h4>
          Simples
        </div>

Tooltip

Elemento original

Exemplo padrão

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Mauris ultrices porta nisl.

        <p>Lorem ipsum dolor sit amet, consectetur <a href="#" rel="tooltip" title="Descrição secundária">adipiscing</a> elit. Mauris ultrices porta nisl.</p>
  • O elemento com o tooltip deverá possuir o atributo rel="tooltip".
  • O texto exibido pelo tooltip deverá estar dentro do atributo title.

Popover

Elemento original

Exemplo padrão

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Mauris ultrices porta nisl.

        <p>Lorem ipsum dolor sit amet, consectetur <a data-content="Lorem ipsum dolor sit amet, consectetur adipiscing elit. Mauris ultrices porta nisl, at tempor nunc commodo eu. Phasellus pulvinar euismod turpis nullam." rel="popover" href="#" data-original-title="Título do popover">adipiscing</a> elit. Mauris ultrices porta nisl.</p>
  • O elemento com o popover deverá possuir o atributo rel="popover".
  • O texto exibido pelo popover deverá estar dentro do atributo data-content.
  • O título do popover deverá estar dentro do atributo data-original-title.
