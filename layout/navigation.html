Grupos de filtros

Grupos de filtros

Use grupos de filtros para juntar múltiplos filtros como um único componente composto. Construá-os com uma série de elementos <a> ou <button>.

Melhores práticas

Nós recomendamos os seguintes guidelines para usar o grupo de filtros e barra de filtros:

  • Sempre use o mesmo elemento em um único grupo de filtros, <a> ou <button>.
  • Não misture filtros de cores diferentes no mesmo grupo de filtros.
  • Use ícones em adição ou ao invés de texto, mas inclua textos no alt e title onde apropriado.

Exemplo padrão

Aqui é como o HTML se parece para um grupo de filtros padrão:

<div class="filter-group">
  <button class="filter">1</button>
  <button class="filter">2</button>
  <button class="filter">3</button>
</div>

Exemplo de barra de filtros

Combine conjuntos de <div class="filter-group"> em um <div class="filter-toolbar"> para componentes mais complexos.

<div class="filter-toolbar">
  <div class="filter-group">
  ...
  </div>
  <div class="filter-group">
  ...
  </div>
</div>

Grupo de filtros do Redu

O exemplo abaixo contrói o componente Filtro descrito na página 24 do G&E 1.3:

<div class="filter-group">
  <button class="filter"><span class="icon-teacher-lightblue_16_18-before">100 professores</span></button>
  <button class="filter filter-active"><span class="icon-tutor-lightblue_16_18-before">100 tutores</span></button>
  <button class="filter"><span class="icon-member-lightblue_16_18-before">100 alunos</span></button>
</div>

Observações:

  • Lembrar de usar a classe no span para exibir a versão azul claro do ícone.
  • Um filtro ativado deve ser marcado com .filter-active.
