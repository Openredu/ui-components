Convites

Tokens

Os tokens de convite são usados quando o usuário está buscando por outros usuários (dentro ou fora do Redu) para enviar um convite de amizade. Existem dois tipos de tokens, um representando um usuário existente no Redu e outro representando um usuário fora do Redu.

Usuário cadastrato no Redu

Quando o usuário está buscando por outros usuários e digita o nome ou e-mail de algum outro usuário que está cadastrado na base de dados do Redu, este token é utilizado.

Exemplo:
Remover Filipe W. Lima 
<!--
  Variáveis:
    USER_PROFILE_URL: a URL do perfil do usuário.
    USER_THUMBNAIL_URL: endereço do thumbnail do usuário tamanho 32x32.
    USER_FULL_NAME: nome completo do usuário.
-->
<div class="invitation-token clearfix">

    <img class="invitation-token-thumbnail" src="USER_THUMBNAIL_URL" alt="USER_FULL_NAME" title="USER_FULL_NAME" width="32" height="32">
  </a>
  <div class="invitation-token-inner">
    <span class="invitation-link-remove icon-close-gray_16_18 text-replacement link-fake">Remover</span>
    <a class="invitation-token-title text-truncate" href="USER_PROFILE_URL" title="USER_FULL_NAME">USER_FULL_NAME</a>
  </div>
</div>

Usuário não cadastrato no Redu

Quando o usuário está buscando por outros usuários e digita um e-mail que não está cadastrado na base de dados do Redu, este token é utilizado.

Exemplo:
Remover
filipewl@hotmail.com(Convidar para o Redu) 
<!--
  Variáveis:
    EMAIL: e-mail do usuário a ser convidado para o Redu.
-->
<div class="invitation-token">
  <div class="invitation-token-inner">
    <span class="invitation-link-remove icon-close-gray_16_18 text-replacement link-fake">Remover</span>
    <div class="invitation-token-title">
      <span class="invitation-token-email text-truncate" title="EMAIL">EMAIL</span><span class="invitation-token-legend legend">(Convidar para o Redu)</span>
    </div>
  </div>
</div>

Convites enviados

Os convites que foram enviados podem ser representados de duas maneiras: para os usuários cadastrados ou não no Redu.

Usuários cadastrados no Redu

Exemplo:
Remover Reenviar convite Filipe W. Lima Convite enviado em 17 Fev, 2012, 16h00 
<!--
  Variáveis:
    USER_PROFILE_URL: a URL do perfil do usuário.
    USER_FULL_NAME: o nome completo do usuário.
    USER_THUMBNAIL_URL: o thumbnail do usuário tamanho 48x48.
    INVITATION_RESEND_URL: a URL para reenviar o convite.
    INVITATION_SENT_TIMESTAMP: a data de quando o convite foi enviado.
-->
<div class="invitation list-mix-inner clearfix">
  <a class="invitation-thumbnail-link" href="USER_PROFILE_URL" title="USER_FULL_NAME">
    <img class="invitation-thumbnail" src="USER_THUMBNAIL_URL" alt="USER_FULL_NAME" title="USER_FULL_NAME" width="48" height="48">
  </a>
  <div class="invitation-inner">
    <span class="invitation-link-remove icon-close-gray_16_18 text-replacement link-fake">Remover</span>
    <a class="invitation-link-resend link-secondary" href="INVITATION_RESEND_URL">Reenviar convite</a>
    <a class="invitation-title text-truncate" href="USER_PROFILE_URL" title="USER_FULL_NAME">USER_FULL_NAME</a>
    <span class="invitation-legend legend">Convite enviado em INVITATION_SENT_TIMESTAMP</span>
  </div>
</div>

Usuários não cadastrados no Redu

Exemplo:
Remover Reenviar convite filipewl@hotmail.com Convite enviado em 17 Fev, 2012, 16h00 
<!--
  Variáveis:
    EMAIL: o e-mail usado no convite.
    INVITATION_RESEND_URL: a URL para reenviar o convite.
    INVITATION_SENT_TIMESTAMP: a data de quando o convite foi enviado.
-->
<div class="invitation list-mix-inner">
  <div class="invitation-inner">
    <span class="invitation-link-remove icon-close-gray_16_18 text-replacement link-fake">Remover</span>
    <a class="invitation-link-resend link-secondary" href="INVITATION_RESEND_URL">Reenviar convite</a>
    <span class="invitation-title text-truncate" title="EMAIL">EMAIL</span>
    <span class="invitation-legend legend">Convite enviado em INVITATION_SENT_TIMESTAMP</span>
  </div>
</div>

Usuários cadastrados no Redu com escolha de papel

Exemplo:
Remover Reenviar convite
Filipe W. Lima(Aluno)
Convite enviado em 17 Fev, 2012, 16h00 
<!--
  Variáveis:
    USER_PROFILE_URL: a URL do perfil do usuário.
    USER_FULL_NAME: o nome completo do usuário.
    USER_THUMBNAIL_URL: o thumbnail do usuário tamanho 48x48.
    INVITATION_ROLE: papel em que o usuário foi convidado.
      Valores: Aluno, Professor, Tutor ou Administrador.
    INVITATION_RESEND_URL: a URL para reenviar o convite.
    INVITATION_SENT_TIMESTAMP: a data de quando o convite foi enviado.
-->
<div class="invitation list-mix-inner clearfix">
  <a class="invitation-thumbnail-link" href="USER_PROFILE_URL" title="USER_FULL_NAME">
    <img class="invitation-thumbnail" src="USER_THUMBNAIL_URL" alt="USER_FULL_NAME" title="USER_FULL_NAME" width="48" height="48">
  </a>
  <div class="invitation-inner">
    <span class="invitation-link-remove icon-close-gray_16_18 text-replacement link-fake">Remover</span>
    <a class="invitation-link-resend link-secondary" href="INVITATION_RESEND_URL">Reenviar convite</a>
    <div class="invitation-title">
      <a class="invitation-title-link text-truncate icon-[member|teacher|tutor|manager]-gray_16_18-before" href="USER_PROFILE_URL" title="USER_FULL_NAME">USER_FULL_NAME</a><span class="invitation-title-role legend">(INVITATION_ROLE)</span>
    </div>
    <span class="invitation-legend legend">Convite enviado em INVITATION_SENT_TIMESTAMP</span>
  </div>
</div>
