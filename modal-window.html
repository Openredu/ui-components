Janela modal

O link que ativará a janela modal deverá possuir o atributo data-toggle="modal". Além disso, deverá ter em seu atributo href, o mesmo id usado na janela modal. É desta maneira que o link saberá qual janela abrir.

A janela modal deverá ter, obrigatoriamente, as classes: .modal com os estilos principais e .hide para mantê-la escondida até ser chamada. Opcionalmente, adicionando a classe .modal-grids-14 aumenta a largura da janela equivalente a 14 grids.

Arquivo

_modal-window.scss

Dependência
  • bootstrap-modal.js
  • bootstrap-modal-redu.js

A estrutura interna da janela é formada por três partes básicas: .modal-header com o cabeçalho, .modal-body com o conteúdo e .modal-footer com o rodapé.

O cabeçalho é formado pelo título da janela h3.header-title e opcionalmente um sub-título .header-subtitle (ex: link). Adicionando a classe .modal-header-gray ele terá a mesma aparência cinza do rodapé.

Na área de conteúdo, adicionando .modal-scroll, uma seta aparecerá indicando a direção possível da barra de rolagem caso o conteúdo seja muito longo para a janela.

Normalmente o rodapé deverá conter algum botão com a opção de fechar a janela. Este elemento deverá ter o atributo data-dismiss="modal".

Exemplos
  • Clique aqui para abrir o exemplo 1 (só texto).
  • Clique aqui para abrir o exemplo 2 (vídeo do YouTube).
  • Clique aqui para abrir o exemplo 3 (muito texto com .modal-scroll).
  • Clique aqui para abrir o exemplo 4 (com .modal-fill-height.modal-grids-14 e .modal-header-gray).
<a href="#MODAL_ID" data-toggle="modal">Link para abrir a janela modal</a>

<div id="#MODAL_ID" class="modal hide [modal-fill-height] [modal-grids-14]">
  <div class="modal-header [modal-header-gray]">
    <!-- IF WANT TO ADD CLOSE BUTTON TO UPPER RIGHT -->
    <span class="modal-close icon-close-big-gray_16_18 text-replacement" data-dismiss="modal" title="Fechar">Fechar</span>
    <!-- END -->
    <h3 class="modal-title">Título</h3>
    <a class="modal-subtitle" href="#">Sub-título</a>
  </div>
  <div class="modal-body [modal-scroll]">
    Conteúdo
  </div>
  <div class="modal-footer">
    <button data-dismiss="modal" class="button-primary">Fechar</button>
  </div>
</div>
